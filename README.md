# GearTalk

GearTalk is a web application for publishing and managing car reviews.
The project is built with ASP.NET Core MVC and Razor Pages.

Key features:
- User registration and login (ASP.NET Core Identity)
- Role-based access (User, Admin, SuperAdmin)
- Admin functionality for managing content
- SQL Server with EF Core

This project is under active development and focuses on structure and functionality.
